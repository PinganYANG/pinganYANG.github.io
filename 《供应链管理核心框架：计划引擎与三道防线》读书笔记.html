供应链管理核心框架：计划引擎与三道防线

序言核心：计划理念的进化

“从‘七分管理’到‘三分技术’的转变，是供应链成熟度跃迁的标志。”

  • 传统痛点：

  • 破局之道：

    • 技术锚点：生命周期管理（新品快速迭代 + 老品精打细算）

    • 空间维度：中心仓（总量平衡）→ 前置仓（精准布货）

    • 业务适配：

      • B2C：应对促销波动 + 短生命周期

      • B2B：管理大客户需求 + 长生命周期

导读精髓：计划是供应链引擎

计划职能的金字塔结构

关键洞见：
“计划的本质是 预见性——想不到则做不到，做到也是高成本。”

供应链三道防线模型

第一道防线：需求计划

  • 方法论：“从数据开始，由判断结束”

    • 数据基石：历史需求分析 → 基准预测（移动平均/指数平滑）

    • 判断修正：销售/市场输入促销/新品因素 → 滚动更新预测

  • 跨职能协作：打破“有数据者无判断，有判断者无数据”困局

第二道防线：库存计划

  • 防区漏洞：

    • 中心仓与前置仓博弈 → 系统库存膨胀

    • 破解：协同设定库存水位（避免互扣人质）

第三道防线：供应链执行

  • 补位逻辑：当预测偏差突破安全库存时

    • 应急手段：紧急采购/调拨/加班生产

    • 成本警示：执行成本通常高于预防成本3-5倍

库存控制三阶段演进

  • 驱动要素： 

    库存 = f(周转周期, 不确定性, 组织行为)  

    • 案例：缩短采购周期30% → 安全库存下降15%

实践路线图

  1. 技术筑基：

    • 清洗历史数据 → 构建预测模型（如Holt-Winters）

    • 设定量化指标（MAPE≤20%，库存周转率≥6）

  2. 管理协同：

    • 月度S&OP会议：平衡销售野心与供应现实

    • 库存责任制：前置仓与中心仓联合KPI

  3. 防线加固：

    • 第一防线：每周预测刷新 + 促销模块校准

    • 第二防线：动态安全库存（按CV波动季度调整）

    • 第三防线：建立应急供应商清单

终极目标：
“用三分技术锁定可重复规律，七分管理应对不可控变量，实现 低库存、高有货 的黄金状态。”

    3. 不脱离结果：通过库存周转率/交付率反推预测质量

1.2 时间序列预测：指数平滑法

方法原理

  • 核心公式：

    Ft+1=αDt+(1−α)Ft(0≤α≤1)

  • α选择逻辑：

初始化的三种方法

  1. 简单初始化：F1=D1

  2. 移动平均法：F1=1k∑i=1kDi

  • 痛点：

    1. 按产品类别分组（如A类α=0.3，C类α=0.7）

1.3 趋势预测：霍尔特双参数平滑法

方法原理

  • 三组件模型：

    水平部分 Lt=αDt+(1−α)(Lt−1+Tt−1)趋势部分 Tt=β(Lt−Lt−1)+(1−β)Tt−1预测值 Ft+n=Lt+n⋅Tt

  • 多期预测风险：n过大可能导致负预测（需设下限约束）

初始化策略

案例演示：
历史数据：周1=100，周2=120 → L1=100,T1=20
周3预测：F3=100+1×20=120

方法选择决策树

本章精华：

  • 移动平均法：稳字当头，期数是关键

  • 指数平滑法：α定乾坤，近期权重大

  • 霍尔特法：水平+趋势双驱动，警惕长期预测陷阱

1.3.4 季节性趋势预测：霍尔特-温特模型

时间序列分解
需求=水平⏟均值+趋势⏟斜率+季节性⏟周期+ε⏟随机噪声
季节性指数四步法

  1. 计算周期均值：

    • 例：Q2三年均值 = (120+130+140)/3 = 130

  2. 计算总均值：

    • 例：四年总均值 = (100+130+150+120)/4 = 125

  3. 求季节指数：

    季节指数=周期均值总均值(例：Q2指数=130/125=1.04)

  4. 预测公式：

    预测值=(水平+n×趋势)×季节指数
短生命周期应对策略

  • 半成品/原材料层预测：利用历史工艺相似性

  • 销售行为预测：建立“压货系数”模型（如季末销量=日常×1.8）
